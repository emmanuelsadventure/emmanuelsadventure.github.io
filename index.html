about me

I am an elementary school student. I love Coding, Web Development, and Game Development, and I want to share my progress with the world.

Welcome to my website! A brief intro about myself: When I was seven years old, my dad introduced me to computer programming. I started by learning Python and coding basics, then I became much more excited when I began developing in HTML, CSS, and Javascript. This is when I started this website. While working on some simple Javascript games, I became eager to know more about Game Development. Currently, I am learning in Unity game engine with the help of my dad.

Note: Only the homepage on my website is using a template. All other webpages are built from scratch.
